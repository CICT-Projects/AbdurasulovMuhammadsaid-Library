## 🖥️ ШАГ 1: Запустите приложение

### Если вы на Windows 🪟
```
Дважды кликните на: start.bat
```
✨ Готово! Два окна терминала откроются автоматически.

### Если вы на Linux или Mac 🐧🍎
```bash
chmod +x start.sh
./start.sh
```

### Если вы хотите запустить вручную 🛠️
**Откройте 2 терминала:**

**Терминал 1 (Backend):**
```bash
cd backend
dotnet run
```

**Терминал 2 (Frontend):**
```bash
cd frontend
npm run dev
```

## 🆘 Что-то не работает?

### Backend не запускается
```bash
cd backend
dotnet build
```
Если много ошибок - удалите папки `bin` и `obj`, затем снова `dotnet run`

### Frontend не загружается
```bash
cd frontend
npm install
npm run dev
```

### API недоступен
- Убедитесь что Backend запущен (проверьте первый терминал)
- Подождите 5-10 секунд после запуска
