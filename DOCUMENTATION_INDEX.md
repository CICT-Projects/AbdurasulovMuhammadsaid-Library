### 📄 Прочие файлы

#### backend/api.http
- Примеры HTTP запросов для тестирования API
- Может использоваться в VS Code или Postman

#### start.bat (Windows)
- Скрипт для автоматического запуска обоих приложений
- Просто двойной клик!

#### start.sh (Linux/Mac)
- Скрипт для автоматического запуска обоих приложений
- `chmod +x start.sh && ./start.sh`
